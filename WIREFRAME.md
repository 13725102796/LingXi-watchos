# 灵息 (LingXi) — Apple Watch 线框图

> 设备：Apple Watch 45mm (198×242 px)
> 设计风格：OLED Dark + 国风水墨毛玻璃
> 触控目标：最小 44×44 pt

---

## 1. 主界面 — 灵境总览 (HomeView)

最核心的日常页面。抬腕即看，2-3 秒可获取关键信息。

```
┌────────────────────────┐
│                        │
│     灵台清明            │ ← 状态文字 (15pt, 青瓷色)
│                        │
│        ╭──╮            │
│      ╭┤  ├╮           │
│     ┤ ╰──╯ ├          │ ← 动态灵莲 (核心视觉)
│      ╰┤  ├╯           │    月白→青瓷 呼吸渐变
│        ╰──╯            │    尺寸约 80×80 pt
│                        │
│   清心境 · 凝神         │ ← 境界名 (20pt, 霜雪色)
│                        │
│   ┌──────────────────┐ │
│   │████████░░░░░░░░░░│ │ ← 修为进度条 (藕荷→鎏金渐变)
│   └──────────────────┘ │    修为 320/600
│                        │
│   ♡ 68    ∿ 52ms      │ ← 心率 + HRV (12pt, 远山黛)
│                        │
└────────────────────────┘
```

### 状态变体

**灵台清明（默认态）：**
- 莲花：月白 → 青瓷 缓慢呼吸 (3s 周期)
- 状态文字：青瓷色 "灵台清明"
- 背景：纯黑 #000000

**心绪微澜：**
- 莲花：月白 → 藕荷 微弱闪烁 (1.5s 周期)
- 状态文字：藕荷色 "心绪微澜"

**心魔暗生（触发弹窗前的过渡态）：**
- 莲花：朱砂红光闪烁 (0.8s 周期)
- 状态文字：朱砂色 "心魔暗生"
- 触觉：2 次短促震动

### 交互
- 上滑 → 切换到 云游历练 页
- 点击莲花 → 展开灵台详情（心率曲线 30min）
- Digital Crown → 无特殊功能（避免误操作）

### 三环设计规范（国风化）

传统 Activity Rings 的国风演绎：
- **动环（Move）**：藕荷色 `#E8B4B4` 弧线，替代红色
- **练环（Exercise）**：青瓷色 `#A8D8D8` 弧线，替代绿色
- **立环（Stand）**：鎏金色 `#C9A96E` 弧线，替代蓝色
- 圆环样式：细线描边（2pt），非填充式
- 背景弧：`#1A1A1A` 低对比度底色
- 完成时：弧线末端微光效果

│        ⬇               │
│                        │
│    清心境·固本          │ ← 新境界 (22pt, 鎏金)
│                        │    金光亮起 + 放大弹入
│                        │

### 动效时序
1. **0s** — 屏幕微光闪白 (0.1s)
2. **0.1s** — 金色粒子从中心扩散 (0.8s)
3. **0.3s** — "突破成功" 文字渐显 (0.5s)
4. **0.6s** — 旧境界名淡出 (0.4s)
5. **1.0s** — 新境界名金光弹入 (0.4s spring)
6. **1.2s** — 触觉反馈：长震 (0.5s)
7. **1.5s** — 贺词渐显 (0.5s)
8. **2.0s** — 按钮出现

## 10. 响应式适配

### 41mm vs 45mm 差异

| 元素 | 41mm (176×215) | 45mm (198×242) |
|------|---------------|----------------|
| 莲花尺寸 | 64×64 pt | 80×80 pt |
| 境界名字号 | 18pt | 20pt |
| 步数大字 | 28pt | 32pt |
| 网格列数 | 3列 (48pt格) | 3列 (56pt格) |
| 按钮高度 | 40pt | 44pt |

### Accessibility 适配

| 设置 | 适配方案 |
|------|---------|
| 大字体 | 隐藏莲花动画，放大文字区域 |
| 粗体文字 | `.bold()` 权重提升 |
| 减少动态效果 | 莲花静态显示，无呼吸动画 |
| VoiceOver | 莲花读作"灵台状态：清明"，进度条读百分比 |
